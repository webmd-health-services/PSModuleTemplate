# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'MODULE_NAME'
Import-Module -Name 'MODULE_NAME'
```

To install privately:

```powershell
Save-Module -Name 'MODULE_NAME' -Path '.'
Import-Module -Name '.\MODULE_NAME'
```
